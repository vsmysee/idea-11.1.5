This inspection reports methods which are highly coupled, i.e. that reference too many other classes. Methods with too high a coupling can be very fragile, and should probably be broken up.

Use the field provided below to specify the maximum acceptable coupling a method might have.

Use the first checkbox below to count references to system classes (the java and javax packages).

Use the second checkbox below to count references to library classes (but not system classes).

This inspection reports logger fields on classes which are not declared static. and final. Ensuring that every classes logger is effectively constant and bound to that class simplifies the task of providing a unified logging implementation for an application. Interfaces, enumerations, annotations, inner classes, and abstract classes are not reported by this inspection.

Use the field below to specify the name of the class which will be used for logging in this project. Fields of that type which are not static. and final will be reported by this inspection.
